# 📊 Sample Store Interactive Dashboard

## 🔍 Project Overview
This project presents an interactive Tableau dashboard built using a sample retail dataset to analyze sales performance, customer behavior, and business trends. The dashboard enables users to explore key metrics through dynamic filters and visual storytelling.

The goal of this project is to derive actionable insights that can help businesses improve decision-making, optimize sales strategies, and enhance customer engagement.

## 📁 Dataset

  - Product categories

---

## 📊 Key Features
- 📌 Interactive filters (Category, Region, Segment)
- 📈 Sales & Profit analysis
- 🧭 Region-wise performance comparison
- 🛒 Category & Sub-category insights
- 📉 Trend analysis over time
- 🔄 Dynamic dashboard navigation

---

## 📌 Key Insights
- Identified top-performing product categories contributing to revenue
- Certain regions generate high sales but lower profit margins
- Seasonal trends impact overall sales performance
- Customer segments show different purchasing behaviors

---

## 🛠 Tools & Technologies Used
- Tableau (Data Visualisation)
- Excel / CSV (Data Source)

Tableau is widely used for creating interactive dashboards and enables users to explore data visually using drag-and-drop features. It supports dynamic filtering, calculated fields, and storytelling through dashboards.

## 💡 Business Recommendations
- Focus on high-profit categories to maximize revenue
- Improve performance in low-profit regions
- Run targeted campaigns for underperforming customer segments
- Leverage seasonal trends for marketing strategies
